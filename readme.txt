== +35 Wordpress Security Features ==

* <strong>Hide Wordpress</strong>: Hide version from all scripts and styles call inside the pages source.
* <strong>Hide wp-login.php</strong>: Create new login link and a 404 error will appear to the default login link.
* <strong>Hide wp-admin</strong>: Only the login link can redirect you to the wp-admin dashboard.
* <strong>Prevent Proxy</strong>: wp-admin Dashboard will allow real connections only.
* <strong>Allow Custom Connections</strong>: wp-admin Dashboard will allow some whitelist countries or/and IPs only.
* <strong>Advanced Firewall</strong>: 'Firewall profiles' is advanced option, You are able to choose High, Medium or Low Level of security.
* <strong>2-Step Verification</strong>:  You are able to choose from many options when you decide to enable 2-Step Verification. Available options: Two factor authentication, Facebook Verification, Four numbers pin code and Security question.
* <strong>reCAPTCHA</strong>: reCAPTCHA is important to save your host resources and your Wordpress safe from spam, You can add it to guest comments, login, register or/and reset password forms.
* <strong>Housekeeping</strong>: Clean your Wordpress, Just delete unused files, comments, revisions, trashes, transient feed or/and relationships.
* <strong>Database Backups</strong>: Manually or Scheduling Database backups, both options are available.
* <strong>Professional Antivirus</strong>: Malware scanner, PHPMussel scanner, File Change Detection, Google Safe Browsing Checker, DB scanner and Spam Lisiting checker.
* <strong>Auto Scan Attachments</strong>: Attachments will be scanned while it is being uploading.
* <strong>Scan New Plugins & Themes</strong>: After you activate your new plugin or theme, Bravo will create a new antivirus process to scan the new files.
* <strong>Brute Force Protection</strong>: The complete security for your and users' passwords by activating Bravo brute force protection options.
* <strong>Blacklist Usernames</strong>: Prevent some usernames from register or log in.
* <strong>Blacklist email provider</strong>: Prevent some email hosting from register like e.g: mail.ru.
* <strong>Min & Max Usernames Length</strong>: Minimum and Maximum chars for registered usernames.
* <strong>Moderate New Members</strong>: New members will be need admin approval before they can use their dashboard.
* <strong>Accounts Protection</strong>: You are able to define the login method (email only or username only or both as default), No weak passwords, Maximum Login Attempts and Whitelist IPs.
* <strong>Who is Online (Live Tracker)</strong>: Watch your online visitors and what are they doing?!, You will be able to see all their browsing details and block/unblock Ips.
* <strong>Inline Visitors Blocking</strong>: Watch your visitors activity using the traffic tracker module and you can block and IP or country when you see unusual activity.
* <strong>Cronjobs (Events Schedules)</strong>: You have full control to set what is the appropriate time to run your events.
* <strong>DB Prefix Wizard</strong>: A wizard was designed to change WP database prefix.
* <strong>Development & Maintenance Mode</strong>: There two modes in order to close your site, Development mode will allow some roles to view site as usual as they know it, but Maintenance mode will close site for all.
* <strong>Bandwidth Saver</strong>: Bravo lets you prevent 'Hotlinking & iFrames', Your hosted images will not show at other websites, and your website will be not shown in iframe.
* <strong>Plugin Self Protection</strong>: You can set password and choose some management roles to give them ability to manage Bravo.
* <strong>Idle Logout</strong>: The plugin will clear the current sessions for logged in users if they hold their accounts without using after (n) seconds, you will choose the duration before forcing them to log in again.
* <strong>Mail Watching</strong>: This tool designed for watching outbound email messages in Wordpress. It can help if someone using backdoor in your blog to send spam emails.
* <strong>Error Pages</strong>: Continuing our efforts to hide Wordpress, We designed this tool to use our 404 templates instead of your theme 404 pages.
* <strong>Log Watching</strong>: If you set the firewall to 'High' and disable WordPress debug, You can watch the error log using or tool.

== Installation ==

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `Bravo WP Ultimate Security` from Plugins page.

The plugin will add a new menu called `Bravo Security`. Goforward using the plugin from here!
